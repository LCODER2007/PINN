================================================================================
                         QUICK START (5 MIN)
================================================================================

1. Install dependencies:
   $ pip install -r requirements.txt

2. Run the pipeline (choose one):
   
   QUICK TEST (30-60 min):
   $ python run_full_pipeline.py --config fastdev.yaml --device cuda
   
   FULL TRAINING (2-4 hours):
   $ python run_full_pipeline.py --config production.yaml --device cuda
   
   INTERACTIVE (Jupyter):
   $ jupyter notebook notebooks/

3. Analyze results:
   $ python analyze_results.py --results_dir results --save_plots

4. Check results:
   - Open: results/figures/velocity_comparison_final.png
   - Open: results/figures/loss_history_final.png
   - Open: results/figures/convergence_analysis.png

================================================================================
                         TROUBLESHOOTING
================================================================================

Q: CUDA out of memory
A: Use CPU: python run_full_pipeline.py --config fastdev.yaml --device cpu

Q: Training is slow
A: Use fastdev config: python run_full_pipeline.py --config fastdev.yaml

Q: Poor convergence
A: Use production config with more epochs

Q: Results don't look right
A: Run analyze_results.py to see detailed convergence plots

See QUICKSTART.md for more troubleshooting tips.
